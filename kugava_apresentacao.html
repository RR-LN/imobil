O que o
KuGava faz

Uma plataforma completa que cobre todo o ciclo imobiliário — da descoberta ao negócio fechado.

01
Listagem de imóveis
Casas, apartamentos e terrenos para venda ou arrendamento. Upload de até 6 fotos por imóvel, descrição detalhada, estatísticas (m², quartos, WC, garagem) e geolocalização por cidade.
Core feature
02
Pesquisa e filtros
Pesquisa full-text por título, descrição e localização. Filtros por tipo de imóvel, tipo de transacção, cidade, faixa de preço e área. Resultados paginados para performance.
Descoberta
03
Chat em tempo real
Mensagens instantâneas entre comprador e vendedor ou agente. Supabase Realtime com WebSockets. Cada conversa está ligada a um imóvel específico com preview no topo do chat.
Realtime
04
Agendamento de visitas
Calendário interactivo com slots de horário disponíveis. Comprador selecciona dia e hora, agente confirma. Notificações push automáticas para ambas as partes após confirmação.
Agenda
05
Sistema de afiliados
Dois modelos em simultâneo: agentes profissionais ganham 2% por negócio fechado, utilizadores comuns ganham 0.5% ou 500 MZN por conversão. Níveis Bronze→Platina com benefícios crescentes.
Diferencial
06
Painel de métricas
Dashboard do afiliado com ganhos mensais, total de indicações, conversões e taxa de conversão. Animação de contagem de valores. Link de indicação personalizável e botão de partilha nativo.
Analytics

Tecnologia

Stack técnico
completo

Escolhido para escalar de 0 a 100.000 utilizadores sem mudar de arquitectura.

Frontend & Mobile
RN
React Native + Expo Core
Framework principal — escreve uma vez, corre em Android, iOS e Web. Expo simplifica builds, updates OTA e acesso a hardware nativo (câmera, notificações, GPS).
AndroidiOSWeb (PWA)Expo RouterEAS Build
RN
React Navigation v6
Navegação entre ecrãs — Stack navigator para fluxos (Home→Detalhe→Chat), Bottom Tabs para a navegação principal, Deep linking para links de afiliados.
StackNavigatorBottomTabsDeep Links
ZST
Zustand
Gestão de estado global — leve, sem boilerplate. Stores separadas por domínio: authStore, propertiesStore, affiliateStore. Persiste sessão com AsyncStorage.
authStorepropertiesStoreaffiliateStore
Autenticação
FB
Firebase Auth Auth
Responsável exclusivamente pela autenticação — email/password, Google Sign-In, tokens JWT. O mais robusto do mercado com SDK nativo para React Native, zero configuração de segurança.
Email/PasswordGoogle OAuthJWT TokensSession Persist
Backend & Infraestrutura
SB
Supabase Backend
Backend completo excepto autenticação — base de dados PostgreSQL, Storage para imagens, Realtime para chat em tempo real, Edge Functions para lógica server-side e RLS para segurança.
PostgreSQLStorageRealtimeEdge FunctionsRLS
PG
PostgreSQL (via Supabase)
7 tabelas relacionais: profiles, properties, conversations, messages, bookings, affiliates, referrals. Índices optimizados, full-text search, backups automáticos.

Distribuição de imagens com cache global. Thumbnails automáticos (?width=300) para o feed. Latência reduzida para utilizadores em Nampula, Beira, Maputo.
